Jalen's Dotfiles
================
Dotfiles for my main desktop system

* Distro: `Arch Linux`
* Window Manager: `i3`
* Bar: `polybar`
* Launcher: `rofi`
* Compositor: `compton`
* Shell: `zsh`
* Terminal Emulator: `termite`
* Editors: `vim`, `sublime text`
* Music Daemon: `mpd`
* Music Client: `ncmpcpp`
* Pandora Client: `pianobar`
* File Manager: `thunar`
* Browser: `firefox`
* Mail Client: `thunderbird`
