Jalen's Dotfiles
================
Dotfiles for my XPS-13 (9343)

* Distro: `Arch Linux`
* Shell: `zsh`
* Window Manager: `i3`
* Bar: `polybar`
* Launcher: `rofi`
* Compositor: `compton`
* Terminal Emulator: `termite`
* Editors: `spacemacs`, `vim`, `sublime text 3`
* Music Players: `ncmpcpp`, `pianobar`
* File Managers: `thunar`, `ranger`
* Browser: `firefox`
* Email Client: `thunderbird`
* Notifications: `dunst`
* Info Script: `neofetch`
