# Julianne's Dotfiles

Dotfiles for my main laptop (Dell XPS 9310).

- Distro: `Arch Linux`
- Shell: `zsh`
- Window Manager: `bspwm`
- Bar: `polybar`
- Launcher: `rofi`
- Compositor: `picom`
- Terminal Emulator: `alacritty`
- Editors: `nvim`, `vscode`
- Music Players: `ncmpcpp`, `spotify`
- File Managers: `ranger`
- Browser: `firefox`
- Notifications: `dunst`
- PDF viewer: `llpp`
