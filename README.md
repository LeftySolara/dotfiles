Jalen's Dotfiles
================
Dotfiles for my main desktop system

##Setup
* Linux Distro: `arch`
* Window Manager: `i3`
* Bar: `i3blocks`
* Launcher: `rofi`
* Compositor: `compton`
* Shell: `zsh`
* Terminal Emulator: `termite`
* Editors: `vim`, `sublime text`
* Music Daemon: `mpd`
* Music Client: `ncmpcpp`
* Pandora Client: `pianobar`
* File Manager: `thunar`
* Browser: `firefox-developer`
* Mail Client: `thunderbird`
