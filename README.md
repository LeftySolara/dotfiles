Jalen's Dotfiles
================
Dotfiles for my XPS-13 (9343)

* Distro: `Arch Linux`
* Window Manager: `i3`
* Bar: `polybar`
* Launcher: `rofi`
* Compositor: `compton`
* Shell: `zsh`
* Terminal Emulator: `termite`
* Editors: `vim, sublime text 3, spacemacs`
* Music Players: `ncmpcpp, pianobar`
* File Managers: `thunar, ranger`
* Browser: `firefox-esr`
* Email Client: `thunderbird`
