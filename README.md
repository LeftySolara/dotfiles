# Jalen's Dotfiles

Dotfiles for my main laptop (Dell XPS 9310).

- Distro: `Arch Linux`
- Shell: `zsh`
- Window Manager: `bspwm`
- Bar: `polybar`
- Launchers: `rofi`
- Compositor: `picom`
- Terminal Emulator: `alacritty`
- Editors: `vim`, `vscode`
- Music Players: `ncmpcpp`
- File Managers: `ranger`
- Browser: `firefox`
- Notifications: `dunst`
